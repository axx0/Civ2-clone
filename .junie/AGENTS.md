# Project Guidelines

This project uses the **Agent OS** framework for maintaining standards, product documentation, and technical specifications.

## Custom Commands

The following custom slash commands are available to manage the Agent OS framework:

- `/discover-standards`: Identify and document new coding standards.
- `/index-standards`: Rebuild the standards index (`agent-os/standards/index.yml`).
- `/inject-standards`: Bring relevant standards into the current context.
- `/plan-product`: Initialize foundational product documentation (mission, tech stack, roadmap).
- `/shape-spec`: Create technical specifications for significant features.

## Storage Locations

- **Standards:** `agent-os/standards/`
- **Product Docs:** `agent-os/product/`
- **Technical Specs:** `agent-os/specs/`
- **Index:** `agent-os/standards/index.yml`
